Modelo de Rede


Cada protocolo é responsável por definir as regras para um objeto específico.


Dividimos então os protocolos em grupos, agrupando cada tipo no que chamamos de CAMADAS



MODELOS

Um conjunto dessas camadas forma o que chamamos de modelo. O modelo basicamente define quais são as camadas necessárias para a montagem de um pacote.


*** Modelo ISO/OSI (Open Systems Interconnection)

O modelo ISO/OSI, como o nome já diz, é um "modelo": isso quer dizer que ele define camadas abstratas a serem tratada (teoria)


 É necessária, portanto, uma implementação que aplique esses conceitos (aplicação).
