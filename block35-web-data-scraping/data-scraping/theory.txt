Data crawling

What is crawling? Web crawling (or data crawling) is used for data extraction and refers to collecting data from either the world wide web or, in data crawling cases – any document, file, etc. Traditionally, it is done in large quantities. Therefore, usually done with a crawler agent.


Você será capaz de:

Realizar requisições web;
Analisar conteúdos HTML para extrair dados;
Aplicar técnicas de raspagem para evitar problemas como bloqueio de acesso;
Armazenar os dados obtidos em um banco de dados.
